نفّذ هذه الأوامر بالترتيب داخل (ryu-env310)
1️⃣ تأكد أنك داخل البيئة
source ryu-env310/bin/activate

2️⃣ خفّض pip و setuptools (مهم جدًا)
pip install "pip<23" "setuptools<65" wheel


تحقق:

pip --version


يجب أن يكون pip 22.x تقريبًا

3️⃣ ثبّت Ryu بدون build isolation
pip install ryu==4.34 --no-build-isolation


⏳ انتظر… يجب أن يثبت بدون Error

4️⃣ اختبر Ryu
ryu-manager --version


✅ المفروض الآن يطبع الإصدار بدون أي crash

5️⃣ شغّل تطبيقك
ryu-manager mohaned_sdn.py
