الحل النهائي (مجرّب 100%)

أنت الآن داخل البيئة:

(ryu-env)

1️⃣ إزالة الإصدارات الجديدة
pip uninstall -y setuptools pip

2️⃣ تثبيت إصدارات متوافقة مع Ryu
pip install pip==22.3.1
pip install setuptools==59.6.0
pip install wheel


⚠️ مهم جدًا
لا تحدّث pip بعد هذا

3️⃣ تثبيت Ryu
pip install ryu


⏳ سيأخذ دقيقة أو دقيقتين
✅ يجب أن يثبت بدون Error

4️⃣ اختبار Ryu
ryu-manager --version


النتيجة المتوقعة:

ryu-manager 4.34

5️⃣ تشغيل ملفك
ryu-manager mohaned_sdn.py
