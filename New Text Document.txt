الخطوات (انسخ نفّذ بالترتيب)
1️⃣ ثبّت Python 3.10
sudo apt install -y python3.10 python3.10-venv python3.10-dev


تحقق:

python3.10 --version

2️⃣ أنشئ venv باستخدام Python 3.10
python3.10 -m venv ryu-env310

3️⃣ فعّل البيئة
source ryu-env310/bin/activate


سيظهر:

(ryu-env310) mohaned@mohaned:~$

4️⃣ ثبّت Ryu (بدون مشاكل)
pip install --upgrade pip
pip install ryu

5️⃣ اختبر Ryu
ryu-manager --version


✅ المفروض يشتغل بدون أي Error

6️⃣ شغّل تطبيقك
ryu-manager mohaned_sdn.py
